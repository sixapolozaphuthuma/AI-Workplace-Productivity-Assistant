# WorkMate AI – Workplace Productivity Assistant

## Project Overview

WorkMate AI is an AI-powered workplace productivity assistant designed to help professionals automate repetitive tasks and improve efficiency. The application uses Artificial Intelligence to generate professional emails, summarize meeting notes, organize daily tasks, and assist with research activities.

The goal of the project is to demonstrate how AI can be applied in real-world workplace scenarios to save time, improve decision-making, and increase productivity.

---

## Features

### 1. Smart Email Generator

* Generates professional emails based on user requirements.
* Supports multiple tones:

  * Formal
  * Informal
  * Persuasive
* Adapts communication style for clients, managers, or team members.

### 2. Meeting Notes Summarizer

* Converts lengthy meeting notes into concise summaries.
* Extracts:

  * Key discussion points
  * Decisions made
  * Action items
  * Deadlines
  * Responsible team members

### 3. AI Task Planner / Scheduler

* Creates structured daily or weekly schedules.
* Prioritizes tasks based on urgency and importance.
* Suggests productivity and time-management strategies.

### 4. AI Research Assistant

* Summarizes articles, reports, and topics.
* Identifies key insights and recommendations.
* Simplifies complex information for easier understanding.

## Setup Instructions

### Prerequisites

Ensure you have the following installed:

* Web Browser (Chrome, Edge, Firefox)
* Visual Studio Code
* Git (optional)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/workmate-ai.git
```

2. Navigate to the project folder:

```bash
cd workmate-ai
```

3. Open the project in Visual Studio Code.

4. Open the `index.html` file in your browser.

### Running the Application

If using a local development server:

```bash
npm install
npm start
```

or

```bash
live-server
```

The application will launch in your browser.
